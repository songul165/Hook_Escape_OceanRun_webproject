Hook Escape: Ocean Run, HTML5 Canvas, CSS ve JavaScript kullanılarak geliştirilmiş 2D bir okyanus kaçış oyunudur. Oyuncu seçtiği balığı kontrol ederek kancalardan kaçar, yemleri toplar, yavru balığı korur ve belirli aralıklarla gelen canavar balığa karşı zehir fırlatarak hayatta kalmaya çalışır.

## Oyun Hakkında

Bu proje, basit bir arcade / survival oyun mantığı üzerine kurulmuştur. Oyuncu okyanus ortamında sürekli ilerleyen bir balığı yönetir. Amaç mümkün olduğunca uzun süre hayatta kalmak, skor toplamak ve seviye atlamaktır.

Oyunda yalnızca kaçış mekanikleri değil, aynı zamanda saldırı, karakter seçimi, seviye sistemi, güçlendirmeler ve müzik kontrolü gibi ek özellikler de bulunmaktadır.

## Özellikler

- HTML5 Canvas ile 2D oyun alanı
- Klavye ile balık kontrolü
- Karakter seçme ekranı
- Farklı özelliklere sahip balıklar
- Kanca engelleri
- Yem toplama ve skor sistemi
- Can sistemi
- En yüksek skor takibi
- Yavru balığı koruma mekaniği
- Canavar balık düşmanı
- Space tuşu ile zehir fırlatma
- Zehir ile canavar balığa hasar verme
- Power-up / güçlendirme sistemi
  - Kalkan
  - Güçlü zehir
  - Yavaşlatma
  - Ekstra can
- Seviye sistemi
- Her seviye geçişinde ekranda büyük seviye bildirimi
- Arka plan müziği açma / kapatma
- Animasyonlu okyanus arka planı
- Mercan, yosun, kabarcık ve deniz tabanı dekorları

## Kullanılan Teknolojiler

Bu proje herhangi bir oyun motoru kullanılmadan geliştirilmiştir.

Kullanılan temel teknolojiler:

- HTML
- CSS
- JavaScript
- HTML5 Canvas API
- LocalStorage
- Audio Element
